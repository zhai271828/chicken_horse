# 联机去瞬移优化报告

日期：2026-05-15

## 问题

- 客机已具备本地预测，但收到主机权威状态后仍会出现明显“瞬移 / 硬拉回”。
- 主机此前只吃每个远端玩家的最新输入，不按逐帧输入队列消费，容易放大主客机分叉。
- 障碍物状态广播频率低于玩家快照，移动平台等动态物体更容易导致客机预测误差。

## 本次优化

- 主机端改为按输入队列逐帧消费远端输入，而不是只覆盖最新输入。
- 客机端保留本地预测与回滚重放。
- 客机收到主机权威状态后：
  - 小误差直接吞掉
  - 中等误差使用视觉偏移平滑回拉
  - 超大误差仍直接硬纠正，避免长期漂移
- 动态障碍物广播频率提升到与玩家快照同档位，减小预测环境差异。

## 说明

- 这次优化目标是“把客机体验拉近主机”，不是一次性做到完全等同本地主机。
- 如果还存在轻微回拉，下一步应继续做：
  - 误差按 X/Y 分轴阈值处理
  - 本地金币/终点视觉预测
  - 主机端更严格的输入时间对齐
