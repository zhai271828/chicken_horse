# 🐔 超级鸡马 - 在线多人派对平台游戏

一个受派对平台游戏启发的在线多人对战原型，包含大厅、准备流程、3-2-1倒计时、鼠标放置陷阱和60秒竞速赛。

## 运行方式

```bash
npm install
npm start
```

在多个浏览器标签页中打开 `http://localhost:3000`。
如果端口 `3000` 被占用，服务器会自动尝试下一个可用端口，并在终端中显示URL。

## 游戏流程

1. 加入大厅房间
2. 选择一个独特的动物角色（兔子、狐狸、小猪、小鸡）
3. 点击「准备」按钮
4. 所有人准备后进入建造阶段
5. 选择一个陷阱道具
6. 移动鼠标预览位置，按 `空格` 旋转方向，点击或按 `E` 放置
7. 所有人放置完毕后，3-2-1倒计时开始
8. 60秒竞速赛开始，冲向终点！

## 操作说明

### 建造阶段
- `1-4`：选择陷阱道具
- `空格`：旋转陷阱方向（上/右/下/左）
- `E`：放置陷阱
- 鼠标移动：预览陷阱位置
- 鼠标点击：放置陷阱

### 竞速阶段
- `A/D` 或 `左/右` 方向键：移动
- `W` 或 `空格`：跳跃
- `R`：重生

## 陷阱道具

| 道具 | 说明 |
|------|------|
| 弹簧跳板 | 踩上去会弹飞玩家 |
| 尖刺地板 | 碰到会击杀玩家 |
| 箭矢炮塔 | 定时发射箭矢 |
| 加农炮塔 | 定时发射炮弹 |
