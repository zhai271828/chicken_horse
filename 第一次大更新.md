## Cloudflare Pages 部署指南

### 项目分析

- **项目类型**: p5.js 游戏（Vite 构建）
- **入口文件**: `index.html` + `src/main.js`
- **构建工具**: Vite
- **当前 base 路径**: `/2026-group-20/main-game/`（需修改为 `/`）

### 部署步骤

#### 方法一：GitHub 连接（推荐）

1. **修改 vite.config.js**
   - 将 `base` 改为 `'/'`
   - 确保 `outDir` 为 `'dist'`

2. **推送到 GitHub**
   ```bash
   git add .
   git commit -m "准备部署到 Cloudflare Pages"
   git push origin main
   ```

3. **在 Cloudflare Dashboard 配置**
   - 登录 https://dash.cloudflare.com
   - 进入 "Workers & Pages"
   - 点击 "Create Application" → "Pages"
   - 连接 GitHub 仓库
   - 设置：
     - **Production branch**: main
     - **Build command**: `npm run build`
     - **Build output directory**: `dist`
     - **Node.js version**: 18 或更高

4. **部署**
   - 点击 "Save and Deploy"
   - 等待构建完成

#### 方法二：直接上传 dist 目录

1. 本地构建：`npm run build`
2. 在 Cloudflare Dashboard 创建项目
3. 直接上传 `dist` 文件夹
